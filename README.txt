  Convert ISCAS85 format to verilog format

  What is it?
  -----------
  The following program converts file in the ISCAS85 format (.isc) to Verilog format (.v)
  
  
  How to compile the program?
  ------------------
  Enter the directory isc2v
  Run the 'make' command
  
  
  How to run the program?
  ------------------
  Usage: ./isc2v iscas_filename
  isc2v is the executable file
  iscas_filename is the name of the ISCAS85 file

  ISCA85 testbench files are located in the directory 'iscas85'
  The output is generated in the file verilog.v
